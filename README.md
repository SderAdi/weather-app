##Weather APP

A simple Weather App built using **HTML, CSS, and JavaScript that fetches real -time weather data from a weather API

## Features

- search weather by city Name
- Shows:
  - City Name
  - Temperature
  - Weather Condition
  - Humidity
- Real-time data using API
- Error handling for invalid city
- Clean and resposive UI

## Technologies

- HTML
- CSS
- JavaScript
- Fetch API
- REST API

## How it Works

1. User Enters a city name
2. App send a request to the weather api
3. API return weather data
4. App display the data on ther screen
